# 🐶 Doggy Style Fashion Pet

## Sobre o Projeto

O Doggy Style Fashion Pet é um site desenvolvido como atividade da disciplina de Desenvolvimento Web. O projeto apresenta uma empresa fictícia especializada em roupas, acessórios e serviços para animais de estimação.

A página foi criada utilizando HTML, CSS, JavaScript e o framework W3.CSS, com o objetivo de aplicar conceitos de estruturação de páginas web, estilização, interatividade e organização de conteúdo.

## Funcionalidades

* Menu lateral de navegação.
* Página inicial com informações da empresa.
* Seção de Serviços e Produtos.
* Slideshow de imagens com navegação manual.
* Área de parceiros comerciais.
* Perguntas frequentes (FAQ).
* Rodapé com informações de contato e redes sociais.
* Layout responsivo utilizando W3.CSS.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* W3.CSS
* Font Awesome

## Estrutura do Site

### Home

Apresenta a empresa, incluindo logotipo, slogan, missão, visão, valores e informações institucionais.

### Serviços / Produtos

Exibe os principais produtos e serviços da empresa por meio de um slideshow interativo, incluindo roupas para pets, acessórios e serviços de banho e tosa.

### Parceiros

Mostra empresas parceiras relacionadas ao universo pet, como áreas de lazer, hospedagem e suplementos.

### FAQ

Contém respostas para as dúvidas mais frequentes dos clientes.

## Uso da Inteligência Artificial

A Inteligência Artificial ChatGPT foi utilizada como consultora técnica para auxiliar na organização do conteúdo, correção de códigos HTML, CSS e JavaScript, desenvolvimento do slideshow, estilização dos elementos visuais e identificação de possíveis erros durante a construção da página.

## Objetivo

Desenvolver uma página web funcional e organizada, aplicando os conhecimentos adquiridos em sala de aula sobre desenvolvimento front-end, usabilidade e design de interfaces.
